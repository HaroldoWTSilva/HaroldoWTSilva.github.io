O Super Milha Otimizeitor é uma aplicação que lhe permite encontrar a mais barata combinação de milhas e dinheiro para sua viagem.

Aplicativos de milhas (ex. Smiles) propõem ao usuário diferentes alternativas de como gastar milhas dinheiro para um mesmo trecho de uma viagem. Além disso, normalmente o usuário já tem algumas milhas e pode comprar mais pelo aplicativo, o que torna a decisão de qual opção escolher não-trivial.
