## O mínimo do básico

Farei aqui algumas observações sobre as probabilidades com dados comuns
de seis faces, conhecidos nos círculos de jogos como *d6*.

Jogar um dado gera um valor. Esse valor é o resultado da jogada do dado.
Um dado comum, honesto, de seis faces deve ter igual chance de gerar
qualquer valor, com probabilidade de *1/6* para cada valor. A um dado desses,
em que os valores possíveis vão de 1 a 6, inclusive, chamaremos de *d6*.

Pode-se jogar um ou mais d6, e seguir uma infinidade de algoritmos para avaliar
o resultado final, ou checar se um certo evento ocorreu. 

Por simplicidade, consideraremos apenas valores possíveis sempre que fizer
sentido.

## Diversos casos

*Jogar um d6 e checar se o valor é igual a X.*
Ex: *d6 = 4*. 

A probabilidade é *1/6*, para todo X.

*Jogar dois d6 e checar se os valores são iguais.*

A probabilidade é *6/36 = 1/6*.

*Jogar dois d6 e checar se ambos os valores são iguais a X.*

A probabilidade é *1/36*. 

*Jogar um d6 e checar se o valor é maior que X.*

A probabilidade é `(6-X)/6`.
Chamemos `dm(X)` a essa função. É uma função linear descrescente. 

*Jogar dois d6 e checar se ambos os valores são maiores que X.*

A probabilidade é `dm(X)^2`. É uma função quadrática decrescente.
Para X=1 a probabilidade é 25 vezes maior do que para X=5.

*Jogar dois d6 e checar se o menor valor é menor ou igual a X.*

A probabilidade é `1 - dm(X)^2`. 

*Jogar dois d6 e checar se a soma dos valores é igual a X.*

A probabilidade é `min( (X-1)/36, (13-X)/36 )`. É uma função de formato
triangular.

## Reflexão

Em jogos de RPG de mesa, é comum usar diversos modelos de dados com números
variados de faces, e até cores variadas para distinguí-los na aplicação de 
certas regras do jogo. 

O autor considera o uso de dados com faces diferentes de seis uma invencionisse
ridícula. Dados de seis faces são comuns e fáceis de obter. 

Jogos de mesa que usam dados para probabilidades deveriam ter suas regras
planejadas para dados de seis faces.
